# AML Editor Frontend

A modern web-based editor for AutomationML (AML) files built with Next.js, React, and TypeScript. This application provides an intuitive interface for viewing, editing, and managing AML hierarchical data structures, more specifically, AML Files of the Q-Metrics.

**Disclaimer**: Frontend is completely vibecoded and a proof of concept, not suitable for production yet.
An improved version can be found in the `develop` branch.
